Software for Large-scale Bound-constrained Optimization

L-BFGS-B is a limited-memory quasi-Newton code for bound-constrained optimization, i.e.
for problems where the only constraints are of the form l<= x <= u. The current release is
version 3.0. The distribution file was last changed on 02/08/11.

(If you have an optimization problem with general constraints, try KNITRO^® )
